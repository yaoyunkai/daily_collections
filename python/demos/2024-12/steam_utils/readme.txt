Steam DB 的 app id, sub id, depots, build id 是什么

    App id
    在 Steam 商店页面看到的每一个独立产品都有一个专属的 App ID。
    app id 有parent app id.

    Sub id
    一个 Sub 可以包含一个或多个 App。它决定了玩家购买后能入库哪些内容。

    Depots
    Depot 是实际存放游戏文件的逻辑分组。
    每个 Depot 也有自己的专属 ID（Depot ID）。一个 App（游戏）通常由一个或多个 Depot 组成。

    build id:
    Steam 为每一次文件更新分配的全局唯一版本号。

    Bundles:
    捆绑包
