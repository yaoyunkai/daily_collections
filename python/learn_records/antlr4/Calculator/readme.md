# ANTLR 表达式解析。

`antlr4 -Dlanguage=Python3 Hello.g4`

```
基于anltr4 设计一个表达式解析工具并计算该表达式的结果，己经准备好了antlr4的相关环境，请使用python实现代码

数字的形式: 支持正负号, 支持整数和小数，比如 2 , -3, -19.03, 19.33, 

1. 支持括号 () , 括号的优先级最高。
2. 加减乘除 + - * / , 乘除的优先级高于加减
3. 支持^ 符号, 该符号是二元的, 2^4 表示2的4次方。优先级应该要高于 加减乘除 
4. 支持% 符号, 两种形式 45% 12.34% 表示百分比相当于除以100. 第二种形式是 8%4 89.4%44.2 求余数。
5. 还要支持 -(5+4) + 5, -67.45^4%, 等写法, 请处理好 () + - * / ^ % 之间的关系。

```
