📐 超椭圆方程： |x/a|^n + |y/b|^n = 1  （Lamé曲线）
当 n = 5 时，曲线在角落处曲率更大、边缘处更平直，形成连续曲率的平滑过渡——这正是 iOS 图标的标志性特征。
